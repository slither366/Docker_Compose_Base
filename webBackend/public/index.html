WEB 2 - BACKEND


Tambien puedes abrir como: localhost/servWeb2

En backend el archivo index siempre esta en public

Para Instalar Laravel sigue los siguientes Pasos

-> Entrar a la Carpeta webBackend

-> borrar la carpeta public

-> Ejecutar un Bash: docker exec -it webRest bash

-> Si estas en Windows debes agregarle al inicio winpty , a veces te avisa


Instalar Laravel con Composer

-> Ejecutar el siguiente comando: composer create-project laravel/laravel

-> te creara un carpeta laravel,

-> debes sacar todo el contenido de la carpeta laravel, para que la carpeta public se quede fuere

La carpeta public es donde se encuentra el index de Laravel


Probar Mysql

-> Ejecutar el siguiente comando: docker exec -it mysql bash

-> En el bash ejecutar: mysql -uroot -proot

-> Ojo -u con el nombre de usuario que declaraste en el docker-compose.yml

-> -p con el password
